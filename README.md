# Orca 런북 (한글 학습 가이드)

[Orca](https://www.onorca.dev) — 여러 AI 코딩 에이전트를 병렬로 실행하는 데스크톱 IDE —
의 **공식 문서([onorca.dev/docs](https://www.onorca.dev/docs))를 한글로 요약·번역한 개인 학습용 런북**입니다.

> 원문 및 이미지 저작권은 Orca(Stably)에 있으며, 이 문서는 학습 목적의 비공식 정리본입니다.
> 세부·최신 내용은 반드시 [공식 문서](https://www.onorca.dev/docs)를 확인하세요.

단일 자립형 HTML 파일(`index.html`)로, 이미지가 모두 임베드되어 있어 오프라인에서도 열립니다.
